# Expense Tracker

A React + TypeScript expense tracking application.

## Features

- Add expenses
- Edit expenses
- Delete expenses
- Category filtering
- Monthly period filtering
- Search expenses
- Sort expenses
- Summary dashboard
- Pie chart analytics
- LocalStorage persistence

## Tech Stack

- React
- TypeScript
- Vite
- Recharts
- CSS
